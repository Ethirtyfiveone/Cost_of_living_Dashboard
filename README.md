# About
This project derived from original tools I used to dial in a precise budget to answer the questions: "What will my rent look like? What are my other monthly costs? What salary can I realistically live on"? I wanted to take the original ideas and develop an interactive Tableau Dashboard. The Dashboard was designed from an Excel concept model, which is a version of my original Workbook. You should be able to access the public dashboard on Tableau, and I made the Excel model available in the Repo.
